# miniproject/시간 부수기

# 1. 프로젝트의 목적과 설명

   Chat GPT 연동 프로젝트
   * 수많은 컨텐츠 들중에서 취향에 맞는 작품을 추천해줄 수 있습니다.
   * 직접적인 질문을 통해 작품에 대한 정보나 탐색이 가능합니다.
